# Assignment 2

## Image Renaming Program

This program renames JPG images in a folder using a sequential naming format.

## Naming Convention

IMG01.jpg
IMG02.jpg
IMG03.jpg

...

IMG50.jpg

## Concepts Used

- File Handling
- OS Module
- Loops
- String Formatting

## How to Run

1. Place all JPG images in a folder.
2. Run the program.
3. Enter the folder path.
4. Images will be renamed automatically.
